# 间距与尺寸系统（ACCESS）

> 以下为 Figma ACCESS 模式下的高度、水平间距、垂直间距、圆角 token，来源于设计系统/间距与尺寸 下的 `Height_ACCESS.tokens.json`、`Horizontal_spacing_ACCESS.tokens.json`、`Vertical_spacing_ACCESS.tokens.json`、`Radius_ACCESS.tokens.json`。  
> 命名建议：`height-{size}`、`spacing-h-{level}`、`spacing-v-{level}`、`radius-{level}`。

---

## 使用说明（建议）

- Height：
  - 默认使用 `height-m`（36），适用于横向细长组件，如 Button、Input、Select 等。
- 水平间距：
  - 文字与 Icon 之间：4（H0）
  - 横向排列的不同组件之间：12（H2）
  - form 组件 label 与输入项之间：24（左右排列场景，H4）
  - 卡片矩阵场景：卡片间距 16/24（H3/H4，依信息量选择）
  - 页面级左右安全边距：**始终 24px（H4）**，禁止因分辨率或内容变化改为其他值
  - 页面/大卡 padding：24（H4）
  - 小模块 padding：16（H3）
- 垂直间距：
  - 表单内上下两项输入之间：24（V4）
  - form 组件 label 与输入项之间：8（上下排列场景，V1）
  - 页面/大卡 padding：24（V4）
  - 小模块 padding：16（V3）
  - 上下两张卡片之间：24（V4）
  - 卡片矩阵场景：卡片间距 16/24（V3/V4，依信息量选择）
- 圆角：
  - 细长小组件（Button/Input 等）：`radius-r1`（6）
  - 卡片：`radius-r2`（10）

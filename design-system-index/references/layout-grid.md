 
# 布局与栅格系统

## 界面适配规则

- **最低适配宽度：1280px**。界面按 1280px 及以上视口设计。
- **小于 1280px 时**：不压缩布局，出现**横向滚动条**，由用户横向滚动查看完整内容。

## 全局规则

- **除侧导外，界面右侧主区均用白色卡片分块**。主内容区内的各个功能区块（表单区、列表区、详情区等）以白色卡片容器划分，卡片之间有24px间距，保持层次清晰。

## 典型布局

按场景分为**表单**、**表格**、**卡片列表**、**数据看板**四类，每类下按是否有侧导区分。

### 表单

#### 带侧导的表单布局

左侧为**侧边导航/步骤条**，右侧主区由**多块白色卡片**纵向排列，卡片间留白（如 24px），适用于多步骤表单或分区块填写的流程。

```
┌─────────┬──────────────────────────────────────┐
│  顶部导航栏（Logo / Tab / 操作 / 用户） 全宽   │
├─────────┼──────────────────────────────────────┤
│         │ ┌──────────────────────────────────┐ │
│  步骤   │ │ 区块标题                          │ │
│  ① ●   │ │ 标签  输入框                       │ │
│  ───    │ │ 标签  输入框                       │ │
│  ②      │ └──────────────────────────────────┘ │
│  ───    │                                      │
│  ③      │ ┌──────────────────────────────────┐ │
│  ───    │ │ 区块标题                          │ │
│  ④      │ │ 标签  输入框  …                   │ │
│         │ └──────────────────────────────────┘ │
│         │ ┌──────────────────────────────────┐ │
│         │ │ [ 主按钮 ]    [ 次要按钮 ]        │ │
│         │ └──────────────────────────────────┘ │
└─────────┴──────────────────────────────────────┘
```

**结构：**
- **顶部导航栏**：最上方通栏，横跨整宽（覆盖侧导+主区），深色或与内容区分界；可含 Logo、Tab、搜索、操作按钮、用户头像等。
- **左侧栏**：浅灰竖条，全高。放置步骤指示器（Stepper/Steps）或章节导航，竖线串联圆点，当前步骤可高亮（如深色圆点）。
- **右侧主区**：多块**白色卡片**自上而下排列，卡片间留白、可带轻微阴影区分层级：
  - **表单区块卡片**：每块卡片内为「区块标题 + 若干标签/输入框」；不同区块（如基本信息、详细设置）分卡展示。
  - **操作栏卡片**：最底部单独一块卡片，主按钮（如「下一步」「提交」）与次要按钮（如「取消」「上一步」）横排。

#### 无侧导表单

无侧边栏，整页为**单块白色卡片**，居中于浅灰背景，适用于单步表单、设置页、创建/编辑页等。

```
┌─────────────────────────────────────────────────┐
│  顶部导航栏（Logo / Tab / 操作 / 用户）          │
├─────────────────────────────────────────────────┤
│  标题 / 说明（可选）                             │
├─────────────────────────────────────────────────┤
│  标签    输入框                                  │
│  标签    输入框                                  │
│  ...                                             │
├─────────────────────────────────────────────────┤
│  [ 主按钮 ]    [ 次要按钮 ]                      │
└─────────────────────────────────────────────────┘
```

**结构：**
- **顶部导航栏**：最上方通栏，全宽；可含 Logo、Tab、操作、用户等。
- **单卡片**：主区仅一块白色卡片，左右留白居中，宽度可参考 720–840px。
- **自上而下**：标题/说明（可选）→ 表单项区（标签左、输入右或上）→ 底部操作栏（主按钮 + 次要按钮）。

### 表格

#### 有侧导表格

左侧为**侧边导航**，右侧主区为**白色卡片**，卡片内为表格能力区（筛选/ Tab + 表头 + 数据行 + 分页），适用于列表页、数据管理页。

```
┌─────────┬──────────────────────────────────────┐
│  顶部导航栏（Logo / Tab / 操作 / 用户） 全宽   │
├─────────┼──────────────────────────────────────┤
│  导航   │  标题 / 搜索栏                        │
│  项 1   ├──────────────────────────────────────┤
│  项 2   │  [ Tab1 ]  Tab2   Tab3                │
│  项 3   ├──────────────────────────────────────┤
│  项 4   │  表头 A  |  表头 B  |  表头 C         │
│         │  数据行…                              │
│         │  …                                    │
│         ├──────────────────────────────────────┤
│         │                     [ 1 ] 2  3  4     │
└─────────┴──────────────────────────────────────┘
```

**结构：**
- **顶部导航栏**：最上方通栏，横跨整宽；可含 Logo、Tab、搜索、用户等。
- **左侧栏**：窄幅竖条，全高，放置导航项或菜单。
- **右侧主区（白色卡片）**：标题/搜索栏 → Tab 或筛选栏 → 表头 → 数据行 → 底部分页。

#### 有侧导表格（标题栏为筛选区）

左侧为**侧边导航**，右侧主区为**白色卡片**，卡片内**标题栏区域为筛选区**（多行筛选项 + 主操作按钮），下方为表头、数据行与分页，适用于带复杂筛选条件的列表页、数据管理页。

```
┌─────────┬──────────────────────────────────────┐
│  顶部导航栏（Logo / Tab / 操作 / 用户） 全宽   │
├─────────┼──────────────────────────────────────┤
│  导航   │  筛选区（多行）                        │
│  项 1   │  [ 筛选项 ] [ 筛选项 ] [ 主操作 ]     │
│  项 2   │  [ 筛选项 ] [ 筛选项 ] …              │
│  项 3   ├──────────────────────────────────────┤
│  项 4   │  表头 A  |  表头 B  |  表头 C         │
│  项 5   │  数据行…                              │
│         │  …                                    │
│         ├──────────────────────────────────────┤
│         │                     [ 1 ] 2  3  4     │
└─────────┴──────────────────────────────────────┘
```

**结构：**
- **顶部导航栏**：最上方通栏，横跨整宽；可含 Logo、Tab、搜索、用户等。
- **左侧栏**：窄幅竖条，全高，放置导航项或菜单。
- **右侧主区（白色卡片）**：**标题栏即筛选区**，可多行排布筛选项（输入框、下拉、日期等）及主操作按钮（如「查询」「重置」）；其下为表头 → 数据行 → 底部分页。

#### 无侧导表格

无侧边栏，整页为**单块白色卡片**，内含表格能力区（筛选/操作按钮 + 表头 + 数据行 + 分页），适用于纯列表、数据查询页。

```

│  顶部导航栏（Logo / Tab / 操作 / 用户）          │
├─────────────────────────────────────────────────┤
│  搜索 / 筛选栏                                  │
│  [ 主操作 ]  [ 次操作 ]  [ 次操作 ]              │
├─────────────────────────────────────────────────┤
│  表头 A  |  表头 B  |  表头 C  |  ...            │
│  数据行…                                         │
│  …                                               │
├─────────────────────────────────────────────────┤
│                          [ 1 ]  2  3  4         │
└─────────────────────────────────────────────────┘
```

**结构：**
- **顶部导航栏**：最上方通栏，全宽；可含 Logo、Tab、操作、用户等。
- **单卡片**：主区一块白色卡片，左右留白居中。
- **自上而下**：搜索/筛选 + 操作按钮 → 表头 → 数据行 → 底部分页。

### 浮层容器

浮层容器用于承载临时性内容（表单、详情、确认等），不占用主页面布局，按触发方式和尺寸分为**抽屉（Drawer）**、**弹窗（Modal/Dialog）**、**气泡（Popover）**。

#### 抽屉容器（Drawer）

从屏幕边缘滑入的面板，通常用于**编辑表单、详情查看、多步操作**等场景，内部可填充表单、列表或其他内容。

```
┌─────────────────────────────────────────────────┐
│  主页面内容（被遮罩层覆盖）                      │
│                        ┌────────────────────────┤
│                        │  标题            ╳     │
│                        ├────────────────────────┤
│                        │ ┌──────────────────┐   │
│                        │ │ 区块标题          │   │
│                        │ │ 标签  输入框      │   │
│                        │ │ 标签  输入框      │   │
│                        │ └──────────────────┘   │
│                        │ ┌──────────────────┐   │
│                        │ │ 区块标题          │   │
│                        │ │ 标签  输入框      │   │
│                        │ │ …                 │   │
│                        │ └──────────────────┘   │
│                        ├────────────────────────┤
│                        │ [ 主按钮 ] [ 次按钮 ]  │
│                        └────────────────────────┤
└─────────────────────────────────────────────────┘
```

**结构：**
- **标题栏**：顶部固定，左侧标题 + 右侧关闭按钮（╳）。
- **内容区**：可滚动，内部可填充**多块白卡**（表单分组、详情区块等）或其他内容。
- **底部操作栏**：固定于底部，主按钮 + 次要按钮横排。
- **宽度**：常见 480–640px（右侧滑入）或自定义。

#### 弹窗容器（Modal / Dialog）

居中浮层，用于**确认操作、简短表单、信息提示**等场景，内部可填充表单、提示文案或其他内容。

```
          ┌────────────────────────────┐
          │  标题                  ╳   │
          ├────────────────────────────┤
          │  标签    输入框            │
          │  标签    输入框            │
          │  …                         │
          ├────────────────────────────┤
          │  [ 主按钮 ]  [ 次按钮 ]    │
          └────────────────────────────┘
```

**结构：**
- **标题栏**：顶部，左侧标题 + 右侧关闭按钮（╳）。
- **内容区**：表单项、提示文案或其他内容；内容过多时可滚动。
- **底部操作栏**：主按钮 + 次要按钮横排，主按钮在左。
- **宽度**：常见 400–600px，根据内容复杂度调整。
- **遮罩**：弹窗外为半透明遮罩，点击遮罩或按 Esc 可关闭（视业务需求）。

#### 气泡容器（Popover）

依附于触发元素的小型浮层，用于**轻量确认、快捷输入、操作菜单**等场景，内部可填充简短表单、选项列表或提示。

```
        ┌──────────────────────┐
        │  气泡标题             │
        ├──────────────────────┤
        │  输入框 / 内容        │
        ├──────────────────────┤
        │       取消    确定    │
        └──────────────────────┘
                 ▼
           [ 触发元素 ]
```

**结构：**
- **标题**（可选）：顶部简短标题。
- **内容区**：输入框、选项列表或提示文案。
- **操作区**：文字链接形式的「取消」「确定」，或小型按钮。
- **箭头**：指向触发元素，明确关联关系。
- **宽度**：紧凑型，常见 240–320px。

## 示例

- ✅ 正确：表单卡片宽度约 800px，居中显示
- ❌ 反例：表单内容满宽铺满导致阅读密度过高

## 修正建议

- 限制表单容器最大宽度并居中
